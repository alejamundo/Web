Introducción


JavaScript es el lenguaje de secuencias de comandos del lado del cliente más popular y ampliamente utilizado. Las secuencias de comandos del lado del cliente se refieren a las secuencias de comandos que se ejecutan dentro de su navegador web. JavaScript está diseñado para agregar interactividad y efectos dinámicos a las páginas web mediante la manipulación del contenido devuelto por un servidor web.

















JavaScript es un lenguaje orientado a objetos y también tiene algunas similitudes en la sintaxis con el lenguaje de programación Java. Pero, JavaScript no está relacionado con Java de ninguna manera.

Lo que puedes hacer con JavaScript

Hay muchas más cosas que puedes hacer con JavaScript.


  • Puede modificar el contenido de una página web agregando o eliminando elementos.
  • Puede cambiar el estilo y la posición de los elementos en una página web.
  • Puede monitorear eventos como hacer clic con el mouse, pasar el mouse por encima, etc. y reaccionar ante ellos
  • Puede realizar y controlar transiciones y animaciones.
  • Puede crear ventanas emergentes de alerta para mostrar información o mensajes de advertencia al usuario.
  • Puede realizar operaciones basadas en las entradas del usuario y mostrar los resultados.
  • Puede validar las entradas del usuario antes de enviarlo al servidor.
